Week 1: Raspberry Pi & Python Programming

1.1 What is Raspberry Pi?

A credit-card-sized computer running Linux, used for electronics and robotics projects.

Popular models: Raspberry Pi 4, Pi 5, Pi Zero

1.2 Basic Setup

  • Install Raspberry Pi OS using Raspberry Pi Imager
  • Enable SSH and VNC
  • Run system updates:
sudo apt update && sudo apt upgrade

1.3 Python GPIO Control

Control an LED connected to GPIO 17:

import RPi.GPIO as GPIO
        import time

        GPIO.setmode(GPIO.BCM)
        LED_PIN = 17
        GPIO.setup(LED_PIN, GPIO.OUT)

        try:
            while True:
                GPIO.output(LED_PIN, GPIO.HIGH)
                time.sleep(1)
                GPIO.output(LED_PIN, GPIO.LOW)
                time.sleep(1)
        except KeyboardInterrupt:
            GPIO.cleanup()
