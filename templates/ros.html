Docker and Containerization for Robotics with ROS 2

Master Docker for scalable, portable, and reproducible ROS 2 applications

Course Overview

This course provides a comprehensive introduction to Docker and containerization tailored for robotics using ROS 2. Learn how to package, deploy, and manage ROS 2 projects in Docker containers with real-world examples and best practices.

  • Duration: 20-30 hours (lectures + hands-on labs)
  • Format: Online or In-person with interactive coding sessions

Prerequisites:

  • Basic Linux command-line skills
  • Familiarity with ROS 2 (Humble or Iron)
  • Basic Python or C++ knowledge

Course Modules

Module 1: Introduction to Docker and Containerization

  • What is Docker and why it's important for robotics
  • Key concepts: containers, images, Dockerfile, Docker Hub
  • Installing Docker on Linux, Windows, macOS
  • Running your first Ubuntu container

Module 2: Running ROS 2 Projects in Docker

  • Benefits of containerizing ROS 2
  • Using official ROS 2 Docker images
  • Networking and volume mounting in Docker

Module 3: Packaging ROS 2 Projects

  • Creating custom Dockerfiles
  • Building and running your containerized ROS 2 package
  • Using docker-compose for multi-container systems

Module 4: Best Practices

  • Optimizing Docker images
  • Security considerations
  • CI/CD integration using GitHub Actions
